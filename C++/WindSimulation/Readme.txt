*****WIND SIMULATION FOR WINDOWS*****
A tech demo for the Nova Game Engine.  Simulates wind blowing though a grassy field.

The aim of the project was to create a wind effect for a real-time application, using as simple a method as possible.  The method should be random, natural, and use no pre-programmed animations.

*****REQUIREMENTS*****
This version of the project is intended to be run on Windows platforms.  It will run on both 32-bit and 64-bit machines.
The project requires OpenGL 4.0 or above.
You may need to install the Microsoft Visual C++ 2017 Redistrubutable on your system before the project will run.
